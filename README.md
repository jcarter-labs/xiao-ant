# XIAO ESP32C3 Antenna Controller

3-antenna selector with pushbutton control.

## Hardware
- XIAO ESP32C3 microcontroller
- ULN2803 relay driver
- 3×12V relays
- 3×LED indicators

## Development Environment
- Ubuntu 25 on Beelink SER8
- Arduino IDE 2.3.7
- ESP32 board package 3.3.5
